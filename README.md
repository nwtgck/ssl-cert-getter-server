## Run Server 

### Way 1: Run server from source 

Run a server from the latest source.

```bash
# Clone this repository
git clone https://github.com/nwtgck/ssl-cert-getter-server.git 
# Go to the directory
cd ssl-cert-getter-server 
# Install dependencies
npm install
# Run a server
npm start
```
